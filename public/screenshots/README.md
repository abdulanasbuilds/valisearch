# ValiSearch Screenshots

Add real PNG screenshots here to replace the HTML mockups on the landing page.

Required files:
- dashboard-overview.png   (full dashboard overview)
- dashboard-kanban.png     (kanban board with tasks)
- dashboard-competitors.png (competitor section)
- dashboard-market.png     (market analysis section)
- hero-analysis.png        (analysis in progress)
- score-display.png        (score circle close-up)

How to take them:
1. Run the app: npm run dev
2. Open http://localhost:5173
3. Type an idea and click Validate
4. Navigate to each section
5. Windows: Win + Shift + S
   Mac: Cmd + Shift + 4
6. Save as the filename above
7. Place in this folder

The landing page will automatically use these real images once they exist here.
